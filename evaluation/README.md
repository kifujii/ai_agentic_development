# 評価チェックリスト

このディレクトリには、各セッションの成果物評価チェックリストが含まれています。

- [セッション1チェックリスト](session1_checklist.md) - AI x IaC基礎実践とAgent開発の理解

## 評価方法

各セッションのチェックリストを使用して、以下の観点から評価を行います：

1. **技術的な完成度**: 構築の品質と完成度
2. **Agent開発の活用度**: Agent機能をどの程度活用できたか
3. **振り返りの質**: 学んだことをどの程度言語化できたか

## 使用方法

1. 各セッションのチェックリストを開く
2. 評価項目を確認する
3. 成果物を確認してチェックリストに記入する
4. フィードバックを提供する

## 注意事項

- チェックリストは評価のガイドラインとして使用してください
- 実際の評価では、参加者の理解度や実装の質も考慮してください
- フィードバックは建設的で具体的なものにしてください
